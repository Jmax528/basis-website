We hebben geleerd dat de blokken in microbits eigenlijk al een sort coderen is maar niet met de hand. Code moet je zo klein en eenvoudig mogelijk proberen te houden en je moet proberen zo min mogelijk code te herhalen als je het kunt voorkomen. Commentaar is belangrijk voor mensen die niet snappen wat er gebeurd en zodat anderen snel je code kunnen begrijpen, dit schrijf je in het engels.
